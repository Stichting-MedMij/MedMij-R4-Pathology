# MedMij-R4-Pathology
This repository contains HL7 FHIR R4 compliant conformance materials for MedMij R4 Pathology, also known as "Pathologie". This project is currently in a pre-publication status and can therefore not be considered stable.

The implementation guide for this project, also in a pre-publication status, can be found on [Simplifier](https://simplifier.net/guide/medmij-r4-pathology-ig).

This repository is maintained by MedMij.

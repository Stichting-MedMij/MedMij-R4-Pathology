# MedMij-R4-Pathology
This repository contains HL7 FHIR R4 compliant conformance materials for MedMij R4 Pathology, also known as "Pathologie" or "PiPP". This project is currently in a pre-publication status and can therefore not be considered stable.

The implementation guide for this project will be published on Simplifier.

This repository is maintained by MedMij.

# Introduction

The functional test data can be found on the underlying pages for each of the different test scenarios. This test data is intended to support interoperability testing. FHIR test material corresponding to the functional test material can be found on [GitHub](https://github.com/Stichting-MedMij/MedMij-R4-Pathology/tree/main/examples).

For mapping between metadata, functional dataset and FHIR, please refer to the {{pagelink: TD, text: technical design, anchor: RelatingFHIRToFunctionalCounterpart}}.

## Structure  
- This IG contains the functional test data from which the FHIR resources are generated.
- The `examples` folder on GitHub contains the FHIR test resources.

## Disclaimer
The FHIR resources are for testing purposes only and may not be entirely correct.
